# Cognitive Experiment — online-only Angular version

Это адаптированная версия существующего Angular-приложения эксперимента. Она построена на исходной архитектуре проекта и переделана под online-only протокол без EEG, baseline-этапов и NIC2-маркеров.

## Текущий flow

- `admin` → consent / данные участника / расширенные инструкции;
- `task?order=1` → блок 1 из 10 вопросов;
- `stage-reminder?order=1` → напоминание о фиксации материалов;
- `nasa-tlx?order=1` → NASA-TLX;
- `strategy?order=1` → описание стратегии и уверенности;
- `task?order=2` → блок 2 из 10 вопросов;
- `stage-reminder?order=2` → напоминание о фиксации материалов;
- `nasa-tlx?order=2` → NASA-TLX;
- `strategy?order=2` → описание стратегии и уверенности;
- `export` → выгрузка результатов.

## Что изменено

- удалён EEG / baseline flow из основного сценария;
- стартовый экран упрощён под online-only протокол;
- убраны заметки экспериментатора и тестовый режим;
- контрбалансировка скрыта от участника и назначается автоматически как `Condition 1` или `Condition 2`;
- результат рандомного назначения записывается в Excel (`Summary`);
- таймер не показывается участнику;
- вопросы продолжают предъявляться в виде изображений;
- NASA-TLX отделён от блока стратегий;
- добавлен отдельный экран с подробным описанием стратегии поиска информации;
- добавлен экран-напоминание после каждого завершённого этапа;
- экспорт включает листы `Summary`, `Answers`, `NASA_TLX`, `Strategies`, `EventLog`.

## Стартовый экран

На стартовом экране теперь используются поля:

- `ID участника` — имя / никнейм / любое значение;
- `Возраст` — полных лет;
- `Пол` — мужской / женский;
- `Образование` — выпадающий список от среднего общего до кандидата наук.

## Инструкции участнику

Исследование проходит в двух этапах: `LLM` и `Search`.

### LLM-блок

- используется языковая модель в отдельной вкладке;
- внутри приложения есть ссылка на DeepSeek Chat;
- после блока участник должен сохранить лог общения с LLM и подготовить его для отправки экспериментатору.

### Search-блок

- используется только обычный браузерный поиск без AI-функций;
- запрещены AI overview, summarize, встроенные ассистенты браузера и сторонние AI-инструменты;
- после блока участник должен сохранить историю поиска за всё время выполнения.

### После завершения эксперимента участник отправляет

- XLSX-файл, выгруженный из приложения;
- лог / экспорт / скриншоты диалога с LLM;
- скриншоты истории поиска.

## Запуск

```bash
npm install
npm run build
```

Для локальной разработки:

```bash
npm start
```

## GitHub Pages

Для деплоя на GitHub Pages обычно требуется корректный `base-href` для имени репозитория:

```bash
ng build --base-href /<repo-name>/
```

После этого можно публиковать содержимое `dist/cognitive-experiment/browser`.
